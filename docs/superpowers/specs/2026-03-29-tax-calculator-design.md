# 役員報酬最適化シミュレーター 設計書

## 概要

一人法人（中小企業）の役員報酬と法人利益の配分を視覚的にシミュレーションし、手取り・税負担の最適化を支援するWebアプリケーション。

## 対象ユーザー

小規模法人（資本金1億円以下）の経営者・役員。汎用ツールとして公開し、個人プリセットは `.local` ファイルで管理（git外）。

## 技術スタック

- **フレームワーク**: Nuxt 3（SSG）
- **UIライブラリ**: shadcn-vue（Radix Vue ベース）
- **スタイル**: Tailwind CSS、ダーク基調
- **パッケージマネージャ**: pnpm
- **デプロイ**: Cloudflare Pages
- **データ永続化**: localStorage（シナリオ保存）

## UI設計

### レイアウト

**2カラム構成**（デスクトップ）→ **1カラム**（モバイル、レスポンシブ）

- **左カラム（入力パネル）**: 基本設定、報酬配分スライダー、控除設定、最適化ゴール
- **右カラム（結果パネル）**: サマリーカード、税負担内訳、ブラケット可視化、シナリオ比較

### デザイントーン

ダーク基調・Fintech風。shadcn/uiのダークテーマをベースに、数値にはネオンカラーのアクセント。

### ブラケット可視化

**2段階表示**:
1. **横バー（常時表示）**: 全体の比率を1本のバーで概観。現在位置が光る線で示される
2. **縦バー（展開時）**: 各ブラケットの税額を棒グラフで表示 + 各段階の説明テキスト

所得税と法人税の両方に適用。

### ユーザーフロー

1. 最適化ゴールを選択（手取り最大化 / 実効税率最小化 / 社保最小化 / トータル手残り最大化）
2. 自動で最適値が算出され、スライダーの初期位置にセット
3. ユーザーが自由にスライダーを動かして探索（リアルタイム更新）
4. 気に入った設定を「シナリオ」として名前付きで保存（localStorage）
5. 複数シナリオを並べて比較

## 入力パラメータ

### 必須入力

| パラメータ | 説明 | デフォルト |
|---|---|---|
| 利益総額 | 役員報酬・社保会社負担控除前の利益（= 売上 - 役員報酬以外の全経費） | 20,000,000円 |
| 月額定期同額給与 | 毎月の役員報酬（万円単位） | スライダーで設定 |
| 事前確定届出給与 | 年額（万円単位）、支給回数（年1〜3回） | 0円 |

**「利益総額」の定義**: UIで「役員報酬を払う前の会社の利益」と明示する。

### オプション入力

| パラメータ | 説明 | デフォルト |
|---|---|---|
| 年齢 | 介護保険の適否判定（40歳以上/未満） | 35歳 |
| 都道府県 | 協会けんぽ料率の決定 | 東京都 |
| 資本金 | 均等割・税率判定 | 100万円 |
| 法人所在地 | 超過税率の適用判定 | 東京23区 |

### 控除設定

**トグル選択 → 自動金額入力 → 手動上書き可能**

| 控除 | 自動入力値 | 備考 |
|---|---|---|
| 基礎控除 | 所得に応じて自動（所得税: 最大95万、住民税: 43万） | 令和7年改正。所得税/住民税で金額が異なる |
| 給与所得控除 | 給与収入から自動計算 | 令和7年〜: 最低65万、上限195万 |
| 社会保険料控除 | 社保個人負担分を全額自動反映 | — |
| 配偶者控除 | トグルON → 38万（一般）/48万（老人） | 本人所得1,000万超で適用不可 |
| 配偶者特別控除 | 配偶者の所得に応じたマトリックス | — |
| 扶養控除 | 一般38万/特定63万/老人48万(58万) | 人数入力 |
| 小規模企業共済等掛金控除 | iDeCo: 月23,000円(年276,000円) | 役員の上限 |
| 生命保険料控除 | 新制度: 最大12万 | 一般/介護/個人年金各4万 |
| 医療費控除 | (医療費 - 10万)、上限200万 | 金額入力 |
| その他控除 | 合計額を自由入力 | — |

## 計算ロジック

### Step 1: 社会保険料の計算

#### 1-a. 月額報酬の社保

```
月額報酬 → 標準報酬月額等級テーブル参照
  ├ 健康保険: 1〜50等級（上限1,390,000円）
  └ 厚生年金: 1〜32等級（上限650,000円）

健康保険料（月額）= 標準報酬月額(健保) × 健保料率
  └ 40歳以上: 健保料率(9.91%) + 介護保険料率(1.59%) = 11.50%
  └ 40歳未満: 健保料率(9.91%)のみ
  → 労使折半

厚生年金保険料（月額）= 標準報酬月額(厚年) × 18.3%
  → 労使折半

子ども・子育て拠出金（月額）= 標準報酬月額(厚年) × 0.36%
  → 全額事業主負担

各月額 × 12 → 年間保険料
```

#### 1-b. 賞与の社保

```
事前確定届出給与 → 標準賞与額（1,000円未満切捨て）
  ├ 健康保険: 上限 年度累計573万円
  └ 厚生年金: 上限 1回あたり150万円

保険料率は月額と同一
子ども・子育て拠出金も賞与に適用（厚年と同じ上限）
```

#### 1-c. 社保合計

```
個人負担 = 月額分個人負担 + 賞与分個人負担
会社負担 = 月額分会社負担 + 賞与分会社負担 + 拠出金合計
社保総コスト = 個人負担 + 会社負担
```

### Step 2: 個人の税金計算

#### 2-a. 給与所得

```
給与収入 = 年間役員報酬
給与所得控除 = テーブル参照（令和7年分〜）
  ├ 〜190万: 65万円（定額）
  ├ 〜360万: 収入×30% + 8万
  ├ 〜660万: 収入×20% + 44万
  ├ 〜850万: 収入×10% + 110万
  └ 850万超: 195万（上限）
給与所得 = 給与収入 - 給与所得控除
```

#### 2-b. 所得控除（所得税用 / 住民税用で別計算）

```
所得控除（所得税用）:
  社会保険料控除（全額） + 基礎控除（6段階、最大95万） + 各種控除

所得控除（住民税用）:
  社会保険料控除（全額） + 基礎控除（43万固定 ※2,400万以下） + 各種控除
  ※ 住民税の控除額は所得税と異なる項目がある
```

#### 2-c. 課税所得

```
所得税の課税所得 = 給与所得 - 所得控除（所得税用）（1,000円未満切捨て）
住民税の課税所得 = 給与所得 - 所得控除（住民税用）（1,000円未満切捨て）
※ 0未満は0
```

#### 2-d. 所得税

```
速算表:
  〜195万:  5%
  〜330万: 10% - 97,500
  〜695万: 20% - 427,500
  〜900万: 23% - 636,000
  〜1,800万: 33% - 1,536,000
  〜4,000万: 40% - 2,796,000
  4,000万超: 45% - 4,796,000

復興特別所得税 = 所得税額 × 2.1%
所得税合計 = 所得税額 + 復興特別所得税（100円未満切捨て）
```

#### 2-e. 住民税

```
所得割 = 住民税の課税所得 × 10% - 調整控除
均等割 = 5,000円（特別区民税3,000 + 都民税1,000 + 森林環境税1,000）
住民税合計 = 所得割 + 均等割

調整控除:
  課税所得200万以下: min(人的控除差額合計, 課税所得) × 5%
  課税所得200万超:   max(人的控除差額合計 - (課税所得 - 200万), 2,500) × 5%
  合計所得2,500万超: 0
```

### Step 3: 法人の税金計算

```
法人所得 = 利益総額 - 年間役員報酬 - 社保会社負担合計

法人税:
  800万以下: 15%（中小法人軽減税率）
  800万超:   23.2%

地方法人税 = 法人税額 × 10.3%

法人住民税:
  法人税割 = 法人税額 × 7.0%（標準税率 ※注1）
  均等割 = 70,000円/年（資本金1,000万以下・従業者50人以下）
  ※注1: 法人税額年1,000万超 or 資本金1億超 → 超過税率10.4%

法人事業税（所得割）:
  400万以下:     3.5%（標準）/ 3.75%（超過 ※注2）
  400〜800万:    5.3%（標準）/ 5.665%（超過）
  800万超:       7.0%（標準）/ 7.48%（超過）
  ※注2: 資本金1億超 or 年所得2,500万超 → 超過税率

特別法人事業税 = 基準法人所得割額（標準税率計算）× 37%
  ※ 超過税率適用法人でも標準税率で計算
```

### Step 4: 結果集計

```
個人手取り = 年間役員報酬 - 社保個人負担 - 所得税合計 - 住民税合計
法人内部留保 = 法人所得 - 法人税等合計
トータル手残り = 個人手取り + 法人内部留保

追加指標:
  個人実効税率 = (所得税 + 住民税) / 年間役員報酬
  法人実効税率 = 法人税等（均等割除く）/ 法人所得 ※損金算入効果反映
  総合税負担率 = (全税金 + 全社保) / 利益総額
  社保負担率 = 社保総コスト / 利益総額
  現在の所得税ブラケット / 次のブラケットまでの余裕額
```

### 最適化アルゴリズム

月額給与を1万円刻みで0〜(利益総額/12)まで走査。各ステップで事前確定届出給与も考慮し、選択されたゴール関数のスコアを計算。最大2,000パターン程度のためブルートフォースで十分。

## 警告・バリデーション

### エラー（計算不可）

- 年間役員報酬 + 社保会社負担 > 利益総額 → 赤字警告
- 月額報酬 < 0 or 事前確定届出給与 < 0 → 入力エラー

### 警告（計算可能だが注意が必要）

| 条件 | 警告メッセージ |
|---|---|
| 月額 = 0円 | 社会保険の加入資格を失う可能性があります |
| 事前確定届出給与のみ | 定期同額給与がない場合の実務リスクがあります |
| 報酬 > 利益の80% | 過大役員報酬と認定されるリスクがあります |
| 報酬 < 利益の10% | 極端に低い報酬は行為計算否認のリスクがあります |
| 月額 > 厚年上限(65万) | 厚生年金保険料は上限に達しています |
| 賞与 > 150万 | 厚生年金の標準賞与額上限超過（年金受給額に影響） |
| 賞与 > 573万 | 健康保険の標準賞与額年間上限超過 |
| 合計所得 > 2,400万 | 基礎控除が減額されます |
| 事前確定届出の支給回数 >= 4 | 年4回以上は報酬として社保計算に含まれます |
| 年間報酬 > 2,000万 | 確定申告が必要です。税理士への相談を推奨します |
| 法人所得 = 0 | 赤字でも均等割70,000円は課税されます |

### 免責表示（常時）

- 本シミュレーションの結果は概算であり、実際の税額を保証するものではありません
- 税制は毎年改正される可能性があります（適用税率: 令和7年度）
- 税務署による過大役員報酬の認定リスクは考慮していません
- 重要な経営判断の際は税理士にご相談ください
- 事前確定届出給与は届出と1円でも異なる金額を支給した場合、全額が損金不算入になります

## 前提条件・制約

- 一人法人（代表取締役のみ）を主な想定
- 協会けんぽ加入（健保組合は未対応）
- 東京都特別区をデフォルト（都道府県切替可能）
- 定常状態の年次計算（初年度特有のズレは考慮しない）
- 雇用保険・労災保険は対象外（役員は原則適用外）
- 消費税は考慮しない
- 他の所得（配当、不動産等）は考慮しない
